# SMM-aero. [Demo](https://lowrider-fn.github.io/smm-aero/)

## Команды
```bash
# старт проекта
yarn start

# сборка проекта
yarn build

# запуск eslint
yarn eslint

# запуск stylelint
yarn stylelint

# проверка типов
yarn types

# переустанавливает зависимости
yarn reinstall

# устанавливает husky и делает файлы хуков исполняемыми
yarn husky
```
